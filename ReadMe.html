utube_Video_Downloader

A Youtube video downloader created using Python.

Project Screenshots:

🧐 Features

Here're some of the project's best features: * Smooth UI * Video Preview * Download Any Youtube Video * Download in your choice of resolution * Handy App * Easy to Use

🛠️ Installation Steps:

1. Installing pipreqs

``` pip install pipreqs ```

2. Run pipreqs

``` pipreqs ```

3. Install Required Libraries

``` pip install -r requirements.txt ```

4. Run the program

``` Run main.py ```
